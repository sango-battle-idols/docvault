How to use

First time for a doc: fill in all three fields and click Generate. Share the permanent link in Discord — it never expires.

Renewing: hit Renew on an existing card (or re-enter the slug). The permanent link stays the same; only the token changes.

Multiple docs: just use different slugs. Each one gets its own permanent link.

⚠️ Active doc list resets if the server restarts. Re-generate tokens after any Vercel re-deploy. See README for persistent storage setup.
